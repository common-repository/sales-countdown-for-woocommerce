== Description ==

WordPress plugin to show countdown on sale product

== Installation ==

1. Install the plugin through Plugins > Add New > Upload or by copying the unzipped package to wp-content/plugins/.
2. Activate the Sales Countdown for WooCommerce plugin through the 'Plugins > Installed Plugins' menu in WordPress.
